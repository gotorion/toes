# toes

toes for tools.
个人常用的工具列表。

## 硬件

### 电脑

- MacBook Pro 16Inch M3 Max 主力工作机器；

### 移动设备

- iPhone 16 Plus 128GB 拍照比较好用；
- OPPO Find N5 大部分时候用微信读书看书用，兼做上班打卡器；
- Apple Watch Series 9 用来订午睡闹钟以及运动记录；

### 配件

- 罗技 MX Master 3 鼠标，大学时候买的二手，现在太能打了以至于无法退役；
- 罗技 G502 鼠标，用来连接台式机用；
- Cherry MX 1.0 机械键盘，在公司写代码用；

## 软件

### 跨平台

- [VSCode](https://code.visualstudio.com/) 默认的代码编辑器和写作软件；
- [滴答清单](https://www.dida365.com/) 我没有分类任务的习惯，只用来作为一个收集箱；
- [微信输入法](https://z.weixin.qq.com/) 简洁无广告，一般用自然码双拼;
- [语雀](https://www.yuque.com/) 平常写一些文章或者记录工作笔记；

### Windows

- [Snipaste](https://www.snipaste.com/) 从大学就在用的截图工具，有免费版本可选；
- [WSL](https://learn.microsoft.com/en-us/windows/wsl/) 不太用得惯Windows开发环境，大部分时候还是使用Ubuntu子系统写代码；
- [PowerToys](https://learn.microsoft.com/en-us/windows/powertoys/) 微软自己开发实用工具集合，基本上只用一个 Alt+Space 快速启动；
- [Windows Terminal](https://learn.microsoft.com/en-us/windows/terminal/) 微软自己开发的终端工具，拿来管理 SSH；
